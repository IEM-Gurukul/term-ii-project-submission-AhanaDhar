# PCCCS495 – Term II Project

## Project Title

Online Food Delivery System (Java OOP Project)


## Problem Statement (max 150 words)

Ordering food manually or through unorganized systems can lead to delays, errors, and poor user experience. This project aims to develop an Online Food Delivery System using Java that allows users to browse restaurants, place orders, and manage deliveries efficiently. The system improves convenience, accuracy, and speed by automating the ordering process and providing a structured way to handle food delivery operations.

## Target User

Customers, restaurant owners, and delivery personnel.

## Core Features

- User registration and login  
- Browse restaurants and menus  
- Add items to cart  
- Place and track orders  
- Order history management  
- Basic admin control for managing food items  


## OOP Concepts Used

- **Abstraction:** Used to hide implementation details in classes like Order and User  
- **Inheritance:** Base class User extended into Customer and Admin  
- **Polymorphism:** Method overriding for different user functionalities  
- **Exception Handling:** Handled invalid inputs and order errors using try-catch  
- **Collections:** Used ArrayList to manage menu items and orders  


## Proposed Architecture Description

The system follows a modular object-oriented design with separate classes for User, Restaurant, Order, and Menu. The main class controls user interaction. Data is stored using collections like ArrayList. The architecture ensures scalability, maintainability, and clear separation of concerns.

## How to Run

1. Clone the repository  
2. Open the project in any Java IDE (VS Code / IntelliJ / Eclipse)  
3. Compile all Java files  
4. Run the Main class  
5. Follow on-screen instructions to use the system
