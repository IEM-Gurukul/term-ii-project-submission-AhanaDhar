ONLINE FOOD DELIVERY SYSTEM
(Java OOP Project)



1. Introduction
The Online Food Delivery System is a Java-based application designed to simplify the process of ordering food. It allows users to browse food items, add them to a cart, and place orders efficiently. The system demonstrates the use of object-oriented programming concepts for better structure and scalability.



2. Objectives
- To automate the food ordering process
- To reduce manual errors
- To provide a user-friendly system
- To implement OOP concepts in a real-world scenario



3. System Features
- User details management
- Restaurant display
- Menu item selection
- Cart functionality
- Order placement system



4. OOP Concepts Used

Abstraction:
Used to hide internal implementation details in classes.

Inheritance:
User class can be extended into different types (Customer/Admin).

Polymorphism:
Methods can behave differently based on object usage.

Encapsulation:
Data is wrapped inside classes like User, Cart, Order.



5. System Design

Classes Used:
- User
- Restaurant
- MenuItem
- Cart
- Order
- Main

Data Structure:
- ArrayList is used to store menu items in cart



6. Working of the System
1. User enters details
2. Restaurant is displayed
3. Menu items are added to cart
4. Order is placed
5. Cart items are shown as order summary


7. Advantages
- Easy to use
- Scalable design
- Organized code structure
- Demonstrates real-world application


8. Conclusion
The project successfully implements an Online Food Delivery System using Java. It demonstrates the use of object-oriented programming concepts and provides a simple and efficient way to manage food orders.
